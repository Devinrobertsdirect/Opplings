# OpplingsEngine Frontend

This is the Phaser.js frontend for OpplingsEngine.

## Setup

```bash
cd frontend
npm install
```

## Run in Development

```bash
npm start
```

## Build for Production

```bash
npm run build
```

## Features
- Top-down map with player movement
- QR code scan button (placeholder)
- Oppling claim and display (placeholder)

## Dependencies
- Phaser
- html5-qrcode
- axios
- Parcel
